<h2>DESCRIPTION</h2>

<em>g.isis3mt</em> Generates an ISIS3 map template file according to the
 current GRASS projection parameters

<h2>NOTE</h2>

The ISIS3 user should use <tt>matchmap=yes</tt> in <tt>cam2mapi</tt>.
