## DESCRIPTION

The *r.mcda.input* module will read a \*rls file output from jMAF, JAMM
or 4eMka2 sofware [Laboratory of Intelligent Decision Support Systems -
Poznan University.](http://idss.cs.put.poznan.pl/) and returns maps of
the rules

*(GRASS python Script)*
