DESCRIPTION

r.slopeunits.create creates a raster layer of slope units. Optionally, a vector map can be created.

EXAMPLE

r.slopeunits.create \
    demmap=dem_italia_isolegrandi@su_test \
    plainsmap=flat \
    slumap=su_tmp \
    thresh=250000 \
    areamin=200000 \
    cvmin=0.25 \
    rf=2 \
    maxiteration=50
