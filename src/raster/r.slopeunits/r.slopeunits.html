DESCRIPTION

r.slopeunits is a GRASS GIS addon toolset that creates, cleans and calculate metrics for slope units. Additionally, optimal input values can be determined. The r.slopeunits toolset consists of currently four modules:
  • r.slopeunits.create: Creates a raster layer of slope units. Optionally, a vector map can be created.
  • r.slopeunits.clean: Cleans slope units layer, e.g. results of r.slopeunits.create.
  • r.slopeunits.metrics: Creates metrics for slope units.
  • r.slopeunits.optimize: Determines optimal input values for slope units.
