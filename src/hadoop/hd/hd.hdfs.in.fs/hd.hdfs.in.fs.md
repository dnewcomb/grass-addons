## DESCRIPTION

*hd.hdfs.in.fs* module allows to PUT data from local filesystem to HDFS

## NOTES

### Usage

The module allows to copy any data from local filesystem to HDFS. For
uploading external GoeJSON files to HDFS it is necessary to modify its
standardized format. The serialization for JSON has several formatting
requirements.
